# FocusFlow iOS (SwiftUI + SwiftData)

This folder now includes a ready-to-open Xcode project.

## Open the project

Open [ios/FocusFlow/FocusFlow.xcodeproj](ios/FocusFlow/FocusFlow.xcodeproj) in Xcode.

## What is included

- Tasks list and editor
- Focus timer with session logging
- Home, Habits, Stats, Settings tabs
- Mood check-ins, reminders, export
- SwiftData models and seed data
